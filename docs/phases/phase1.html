The AI Workflow

Objective: Move from manual code typing or autocomplete to an agentic workflow using the Plan-Build-Review loop.

The Project Goal

Throughout this workshop, we'll be building a retail sales analytics script. Given a dataset of online retail transactions (the UCI Online Retail dataset), our goal is to:

  • Ingest and validate the transaction data defensively
  • Calculate total revenue by country
  • Detect anomalies such as returns and missing customer IDs
  • Write a report file to reports/analysis_report.txt with revenue by country and anomaly details

This is the real problem we'll solve - not code for code's sake. Every phase of this workshop builds toward a working src/analysis.py script that delivers these results.

Autocomplete vs. Agents

Traditional autocomplete suggests the next few characters. Agentic AI understands context, reasons about your intent, and can execute multi-step tasks.

Old Way: Autocomplete

  • Suggests next line of code
  • No understanding of project context
  • You drive every decision
  • Reactive, not proactive

New Way: Agentic Workflow

  • Understands entire codebase
  • Plans before implementing
  • Executes multi-file changes
  • Iterates based on feedback

The Core Loop: Plan-Build-Review

This is the rhythm you'll use throughout the workshop and in your daily work:

1. Plan (Plan Agent)

Use the Plan Agent to design your approach before writing any code:

  • Describe what you want to build
  • Ask for an outline of functions/modules needed
  • Clarify edge cases and requirements

2. Build (Agent Mode)

Use Agent Mode to implement across multiple files:

  • Reference your plan in the prompt
  • Let the agent scaffold the structure
  • Review and accept changes file by file

3. Review (Ask Agent + Git Hooks)

Combine Ask Agent soft checks with hard Git hooks:

  • Open the Ask Agent and prompt it to review your code for bugs, style issues, or complexity violations
  • Run pre-commit for enforcement
  • If hooks fail, feed the error back to the agent and iterate
Key Insight: The AI is your pair programmer, but your Git hooks are the code reviewer that never sleeps. Trust the hooks over the AI's assurances.
