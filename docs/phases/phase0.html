Phase 0 5 min

Preflight Check

Objective: Verify the environment before we start.

Before we begin, please ensure you have everything set up correctly:

  1. VS Code is open - We'll be using VS Code throughout this workshop.
  2. GitHub Copilot extension is installed and signed in - Check for the Copilot icon in the bottom right corner of VS Code. Click it to verify you're authenticated.
  3. Terminal is open - Open a terminal in VS Code via Terminal → New Terminal (or Ctrl+`).
  4. Python is recognized - Type python --version in your terminal. You should see Python 3.10 or higher.
  5. Git is recognized - Type git --version in your terminal. You should see a version number.
Troubleshooting: If any of these checks fail, raise your hand and we'll help you get set up before we proceed.

Once all checks pass, you're ready to begin the workshop!
